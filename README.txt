Instructions:
-------------
1. Install packages
    - pip install opencv-python
    - pip install opencv-contrib-python
    - pip install pyautogui
    - pip install matplotlit
    - pip install pyscreenshot
    
2. Run Jupyter notebook from terminal
3. Open .ipynb notebook on browser
4. Open game like: https://cdn-factory.marketjs.com/en/karate-chop-kick/index.html
5. Position the game windows accordingly 
6. Since the every PC screen is different so set the cordinates used in the code accordingly

7. To RUN the Game execute every cell of notebook
8. Press "q" to stop the game
